# Creator League

A gamified video-editing competition platform for teenagers to track challenges, submissions, peer ratings, and leaderboards.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

## About

Creator League is a platform designed to gamify video creation and editing for content creators. Join competitions, submit entries, and climb the leaderboards.
